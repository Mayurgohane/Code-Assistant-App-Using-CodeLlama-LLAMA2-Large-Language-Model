# Code-Assistant-App-Using-CodeLlama-LLAMA2-Large-Language-Model

This project provides a web-based interface for interacting with the CodeMaster model using a local API. It leverages Gradio to create a simple and intuitive user interface, allowing users to input prompts and receive generated responses. The project also utilizes `ollama` for model handling and management.

Features
•	User-Friendly Interface-A clean and straightforward interface built with Gradio.
•	Persistent History-Keeps track of the conversation history, allowing for context-aware responses.
•	Local API Integration- Communicates with a locally hosted API to generate responses based on user input.
•	Model Management -Utilizes `ollama` for efficient model handling and management.
